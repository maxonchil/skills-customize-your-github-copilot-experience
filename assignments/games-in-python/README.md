
# 🎮 Hangman Game Challenge

## 🎯 Objective

Create a classic word-guessing game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Develop a complete Hangman game that randomly selects a word and allows players to guess letters. The game should track the player's progress, manage incorrect guesses, and determine win/loss conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in underscore format (_ _ _)
- Track the number of incorrect guesses remaining
- End the game when the word is guessed or all attempts are exhausted
- Display appropriate win/lose messages with final results
